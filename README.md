# CivicTrack

Smart Community Complaint Management Platform

## Features

- JWT Authentication
- Role-Based Access Control
- Citizen Dashboard
- Officer Dashboard
- Admin Dashboard
- Complaint Management
- Officer Assignment
- Complaint Analytics

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- bcryptjs

## Installation

### Backend

cd backend

npm install

npm run dev

### Frontend

cd frontend

npm install

npm run dev

## User Roles

- Citizen
- Officer
- Admin

## Workflow

Citizen Creates Complaint
↓
Admin Assigns Officer
↓
Officer Resolves Complaint
↓
Citizen Tracks Status
